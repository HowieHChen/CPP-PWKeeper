# CPP-PWKeeper
## 说明
Cpp大作业  
密码管理程序  
使用QT 6.1.0  

#### 基础UI  
**BaseDialog** 自定义对话框  
**BasePage** 自定义窗口  
**SwitchControl** 滑动开关  
  
#### 基础功能
**INIService** 对INI文件进行操作  
**UserProfile** 用户管理  
**PasswordManage** 密码管理  
  
#### 上层应用
**SignInPage** 登录界面  
**SignUpPage** 注册界面  
**ForgetPWPage** 重置密码界面  
**MainPage** 应用主界面
